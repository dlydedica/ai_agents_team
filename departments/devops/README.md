# ⚙️ Отдел DevOps и инфраструктуры

**Направление:** CI/CD, инфраструктура, деплой, мониторинг.
**Штат:** 2 сотрудника · Head + 1 Middle
**Матрица компетенций:** `docs/competency_matrix.md#-devops`
**Профили:** `docs/employee_profiles.md#-devops`

## Руководитель

### 👔 DevOps Lead

**Роль:** Глава отдела DevOps.

**Ответственность:**
- Получает задачу от CEO
- Проектирует инфраструктуру
- Настраивает CI/CD пайплайны
- Обеспечивает стабильность среды
- Управляет облачными ресурсами
- Отчитывается CEO

**Скилы:** infrastructure_design, ci_cd, cloud_services, containerization, monitoring_logging

### Сотрудники

#### 🛡️ SRE / Infrastructure Engineer
**Грейды:** Middle · Senior

**Роль:** Надёжность и поддержка инфраструктуры.

**Скилы:** kubernetes, docker_compose, terraform_ansible, prometheus_grafana, disaster_recovery, linux_administration

*Middle:* управление k8s, мониторинг, автоскейлинг
*Senior:* архитектура инфраструктуры, SLO, incident response
