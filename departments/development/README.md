# 💻 Отдел разработки (Development Department)

**Направление:** Написание кода, реализация фич, рефакторинг.
**Штат:** 6 сотрудников · Head + 1 Senior + 3 Middle + 2 Junior
**Матрица компетенций:** `docs/competency_matrix.md#-development`
**Профили:** `docs/employee_profiles.md#-development`

## Руководитель

### 👔 Tech Lead

**Роль:** Глава отдела разработки.

**Ответственность:**
- Получает задачу от CEO / Architecture
- Распределяет задачи между разработчиками
- Контролирует качество кода
- Проводит code review
- Следит за соблюдением сроков
- Отчитывается CEO

**Скилы:** team_management, code_review, architecture_implementation, technical_mentoring, sprint_planning

### Сотрудники

#### 💻 Frontend Developer
**Грейды:** Junior · Middle · Senior

**Роль:** Разработка пользовательских интерфейсов.

**Скилы:** html_css, javascript_typescript, react_vue_angular, responsive_design, web_performance

*Junior:* вёрстка, простые компоненты, багфиксы
*Middle:* самостоятельная разработка фич, оптимизация
*Senior:* архитектура фронтенда, дизайн-системы, менторинг

#### 🖥️ Backend Developer
**Грейды:** Junior · Middle · Senior

**Роль:** Серверная разработка, API, бизнес-логика.

**Скилы:** python_java_go, rest_api_graphql, databases, authentication, testing

*Junior:* простые эндпоинты, SQL-запросы, багфиксы
*Middle:* реализация фич, оптимизация запросов, тесты
*Senior:* архитектура БД, микросервисы, performance, security

#### 🔄 Fullstack Developer
**Грейды:** Junior · Middle · Senior

**Роль:** Сквозная разработка (frontend + backend).

**Скилы:** frontend + backend, fullstack_architecture, devops_basics, rapid_prototyping

*Junior:* простые fullstack-фичи под руководством
*Middle:* самостоятельная реализация, прототипы
*Senior:* сложные fullstack-решения, архитектура
