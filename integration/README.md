# 🔗 Интеграция AI Agents Team в ваш проект

Этот пакет позволяет добавить **Оркестратора AI-команды** в Copilot Chat вашего проекта.

## Как это работает

После интеграции в вашем проекте появится кастомный агент **🧠 CEO — Оркестратор AI-команды**, доступный из выпадающего списка агентов в Copilot Chat.

Вы просто описываете задачу, а Оркестратор:
1. Анализирует задачу
2. Определяет, какие отделы AI-команды нужны
3. Поочерёдно делегирует подзадачи отделам
4. Собирает результат и отдаёт вам

## Способы интеграции

### Способ 1: Автоматическая (рекомендуется)

```bash
# Из директории ai_agents_team
python integration/integrate.py /путь/к/вашему/проекту
```

### Способ 2: Ручная

1. Скопируйте `integration/orchestrator.agent.md` в `.github/agents/orchestrator.agent.md` вашего проекта
2. Убедитесь, что `ai_agents_team` находится в корне проекта
3. Откройте проект в VS Code

### Способ 3: Git submodule

```bash
# В корне вашего проекта
git submodule add https://github.com/dlydedica/ai_agents_team.git ai_agents_team
python ai_agents_team/integration/integrate.py .
```

## Требования

- Visual Studio Code
- GitHub Copilot Chat (расширение)
- Проект открыт как workspace в VS Code

## Использование

После интеграции:

1. Откройте **Copilot Chat** (`Ctrl+Shift+I` / `Cmd+Shift+I`)
2. В выпадающем списке агентов выберите **🧠 CEO — Оркестратор AI-команды**
3. Опишите задачу, например:
   ```
   Создай микросервис для сокращения ссылок на FastAPI
   ```
4. Оркестратор покажет план и начнёт последовательно выполнять шаги

## Примеры задач

- "Создай REST API для управления пользователями с JWT-аутентификацией"
- "Добавь CI/CD пайплайн в GitHub Actions"
- "Напиши тесты для модуля payment.py"
- "Спроектируй архитектуру микросервисного приложения"
- "Обнови README и добавь документацию по API"
