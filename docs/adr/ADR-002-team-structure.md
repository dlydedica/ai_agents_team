# ADR-002: Структура команды

**Статус:** Принято
**Дата:** 2026-06-28
**Автор:** 🏗️ Architecture

## Контекст
Необходимо организовать AI-агентов в команду, способную выполнять задачи разработки полного цикла.

## Решение
Использовать **корпоративную иерархию**:
- CEO → Heads (руководители 13 отделов) → Сотрудники

Отделы покрывают полный цикл: Product → Architecture → Development → QA → DevOps → Design → Docs → HR → Security → Data → R&D → Legal → Marketing.

## Альтернативы
- **Плоская структура** — CEO напрямую управляет 35+ агентами (перегрузка)
- **Матричная структура** — сложна для AI-агентов, нет чётких handoff

## Последствия
- Каждый Head отвечает за свой отдел
- Handoff между отделами — через MCP-сервер
- 13 отделов = 13 subagent'ов
