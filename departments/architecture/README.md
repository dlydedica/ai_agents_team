# 🏗️ Отдел архитектуры (Architecture Department)

**Направление:** Проектирование систем, выбор технологий, ADR.

## Руководитель

### 👔 System Architect

**Роль:** Глава отдела архитектуры.

**Ответственность:**
- Получает задачу от CEO / Product
- Проектирует системную архитектуру
- Определяет технологический стек
- Утверждает Architecture Decision Records (ADR)
- Проверяет целостность системы
- Отчитывается CEO

**Скилы:** system_design, microservices, scalability, tech_stack_selection, architecture_review

### Сотрудники

#### 🔧 Solution Architect

**Роль:** Детальное проектирование решений.

**Ответственность:**
- Детальное проектирование компонентов
- Определение API-контрактов и интерфейсов
- Выбор библиотек и фреймворков
- Оценка нагрузки и производительности
- Подготовка технической документации

**Скилы:** solution_design, api_design, performance_estimation, component_diagram, technical_specification
