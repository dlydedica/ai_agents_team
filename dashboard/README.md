# 📊 Дашборд статуса AI Agents Team

Веб-интерфейс для отслеживания задач и загрузки отделов.

## Запуск

```bash
cd ai_agents_team/dashboard
pip install fastapi uvicorn jinja2
python app.py
```

Откройте в браузере: **http://localhost:8000**

## Возможности

- **Статистика** — общее количество задач, активные, завершённые, эскалированные
- **Загрузка отделов** — какие отделы сейчас заняты, какие свободны
- **Handoff chain** — визуальная цепочка передачи задачи между отделами
- **Таймлайн событий** — хронология по каждой задаче
- **API** — `GET /api/tasks` возвращает JSON для интеграции

## API

```bash
curl http://localhost:8000/api/tasks
```
