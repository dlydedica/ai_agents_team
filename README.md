# AI Agents Team 🤖

**Команда AI-разработчиков**, способная автономно анализировать задачи, распределять роли и выполнять проекты любой сложности.

## Концепция

Данный репозиторий содержит **команду виртуальных AI-агентов**, каждый со своей специализацией. При получении задачи команда самостоятельно:

1. **Анализирует** задачу и определяет, какие роли/скилы потребуются
2. **Планирует** архитектуру и разбивает задачу на подзадачи
3. **Делегирует** подзадачи подходящим агентам
4. **Выполняет** — каждый агент делает свою часть
5. **Проверяет** — код проходит ревью и тестирование
6. **Собирает** результат в единое решение

## Состав команды

| Агент | Роль | Ключевые скилы |
|-------|------|----------------|
| 🧠 **Orchestrator** | Лид / Менеджер | Анализ задач, планирование, делегирование, контроль, синтез результата |
| 📐 **Architect** | Архитектор | Проектирование, выбор технологий, структура проекта, проверка целостности |
| 💻 **Developer** | Разработчик | Написание кода, реализация фич, рефакторинг |
| 🔍 **Reviewer** | Ревьюер / QA | Code review, тестирование, поиск багов, проверка качества |
| 📖 **Documenter** | Технический писатель | Документация, README, API-спецификации, комментарии |

## Как это работает

```
┌─────────────────────────────────────────────────────────┐
│                      Задача                             │
└──────────┬──────────────────────────────────────────────┘
           │
           ▼
┌──────────────────┐
│   Orchestrator   │  Анализирует задачу, определяет состав
│   (Lead Agent)   │  и порядок работы агентов
└──────┬───────────┘
       │
       ├──► Architect ────► Developer ────► Reviewer
       │                                      │
       │         ┌────────────────────────────┘
       │         ▼
       │    Documenter
       │         │
       └─────────┼────────────────────────────────────────┘
                 ▼
       ┌──────────────────┐
       │   Orchestrator   │  Собирает результат, формирует отчёт
       │   (результат)    │
       └──────────────────┘
```

## Быстрый старт

1. Склонируйте репозиторий в ваш проект:
   ```bash
   git submodule add <url> ai-team
   ```
2. Поместите задачу в `tasks/` или передайте команде через CLI.
3. Запустите команду:
   ```bash
   python team.py run "Описание задачи"
   ```
