# 🛡️ Отдел информационной безопасности (Security / InfoSec)

**Направление:** Безопасность приложений, инфраструктуры, compliance.

## Руководитель

### 👔 Security Lead / CISO

**Роль:** Глава отдела безопасности.

**Ответственность:**
- Аудит безопасности приложений и инфраструктуры
- Управление уязвимостями и инцидентами
- Security code review
- Compliance (GDPR, SOC2, ISO 27001, PCI DSS)
- Политики безопасности и best practices
- Отчитывается CEO

**Скилы:** security_audit, vulnerability_management, incident_response, compliance, security_policy, risk_assessment

### Сотрудники

#### 🔒 Security Engineer

**Роль:** Техническая безопасность.

**Ответственность:**
- SAST/DAST сканирование, пентесты
- Настройка WAF, IDS/IPS
- Безопасность CI/CD пайплайнов
- Управление секретами и доступом

**Скилы:** penetration_testing, sast_dast, network_security, container_security, secrets_management, iam

#### 📋 Compliance Officer

**Роль:** Соответствие стандартам и регуляторам.

**Ответственность:**
- Аудит compliance (GDPR, 152-ФЗ, SOC2)
- Подготовка документации для сертификации
- Оценка рисков третьих сторон
- Политики обработки данных

**Скилы:** gdpr, soc2, iso27001, risk_management, data_protection, policy_writing

## Когда привлекается
- Любой код перед релизом (security review)
- Настройка инфраструктуры (безопасность сети, доступа)
- Обработка персональных данных (PII)
- Инциденты безопасности
- Аудит зависимостей (Dependabot, Snyk)
