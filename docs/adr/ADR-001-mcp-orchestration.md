# ADR-001: Выбор технологии оркестрации

**Статус:** Принято
**Дата:** 2026-06-28
**Автор:** 🏗️ Architecture

## Контекст
Потребовалось координировать работу 13 отделов AI-агентов. Нужна система управления задачами, handoff и эскалацией.

## Решение
Использовать **Model Context Protocol (MCP)** с Python SDK.

MCP обеспечивает:
- Единый протокол для AI-агентов
- Поддержку инструментов (tools), ресурсов, промптов
- Stdio-транспорт — не требует сети, работает локально

## Альтернативы
- **REST API** — избыточно для локальной оркестрации, требует HTTP-сервер
- **Message Queue (RabbitMQ, Redis)** — тяжеловесно для данной задачи
- **CLI-команды** — нет единого протокола, сложно расширять

## Последствия
- Зависимость от MCP SDK (Python), нужно следить за обновлениями
- Простая интеграция с VS Code через `.vscode/mcp.json`
- 9 инструментов покрывают все сценарии
