# 📊 Отдел данных и аналитики (Data & Analytics)

**Направление:** Data Engineering, Data Science, BI-аналитика.
**Штат:** 4 сотрудника · Head + 2 Middle + 1 Junior
**Матрица компетенций:** `docs/competency_matrix.md#-data`
**Профили:** `docs/employee_profiles.md#-data`

## Руководитель

### 👔 Data Lead / Head of Data

**Роль:** Глава отдела данных.

**Ответственность:**
- Стратегия работы с данными
- Проектирование пайплайнов данных
- Управление data-командой
- Определение метрик и KPI
- Отчитывается CEO

**Скилы:** data_strategy, pipeline_architecture, team_management, kpi_design, data_governance

### Сотрудники

#### 🔧 Data Engineer
**Грейды:** Junior · Middle · Senior

**Роль:** Построение пайплайнов и инфраструктуры данных.

**Ответственность:**
- ETL/ELT пайплайны
- Настройка хранилищ данных (DWH, Data Lake)
- Оптимизация запросов
- Качество данных

**Скилы:** etl, apache_spark, airflow, dbt, sql, data_warehouse, big_data

*Junior:* простые ETL, SQL-запросы
*Middle:* пайплайны, DWH, оптимизация
*Senior:* архитектура данных, big data, real-time

#### 🧪 Data Scientist / ML Engineer
**Грейды:** Middle · Senior

**Роль:** Модели машинного обучения, анализ данных.

**Ответственность:**
- Построение ML-моделей
- A/B тестирование
- Feature engineering
- Эксперименты

**Скилы:** machine_learning, python, pytorch_tensorflow, statistical_analysis, experiment_design, model_deployment

*Middle:* обучение моделей, A/B тесты, пайплайны
*Senior:* архитектура ML, MLOps, research

#### 📈 BI Analyst
**Грейды:** Junior · Middle · Senior

**Роль:** Дашборды, отчёты, визуализация.

**Ответственность:**
- Построение дашбордов
- Анализ метрик продукта
- Ad-hoc отчёты
- Визуализация данных

**Скилы:** tableau_powerbi, superset, sql, data_visualization, product_analytics, reporting

*Junior:* простые дашборды, SQL-запросы
*Middle:* сложные дашборды, анализ метрик, отчёты
*Senior:* data storytelling, стратегия аналитики

## Когда привлекается
- Проекты с обработкой больших данных
- ML-функциональность
- Аналитика продукта (метрики, дашборды)
- ETL-пайплайны
- A/B тестирование
