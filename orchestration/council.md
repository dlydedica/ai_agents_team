# 🤝 Совет отделов (Cross-department Council)

**Роль:** Координационный орган для межотдельского взаимодействия.

## Когда созывается
- Когда задача требует участия 3+ отделов
- Когда есть конфликт между отделами
- Когда нужно принять архитектурное решение, затрагивающее несколько отделов
- Когда CEO требуется коллективная экспертиза

## Участники
- **CEO** (председатель)
- **Heads задействованных отделов**
- При необходимости — ключевые сотрудники

## Формат совета

```
🎙 CEO: "Задача требует coordination между Product, Architecture и Development.
       Какие вопросы нужно решить?"

🗣 Head of Product: "Нужно уточнить требования к MVP..."
🗣 Head of Architecture: "Предлагаю следующий подход к архитектуре..."
🗣 Head of Development: "Оцениваю срок в 3 дня при таком подходе..."

📋 Итог: согласованный план действий
```
