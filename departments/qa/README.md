# 🧪 Отдел тестирования (QA Department)

**Направление:** Обеспечение качества, тестирование, автоматизация.

## Руководитель

### 👔 QA Lead

**Роль:** Глава отдела тестирования.

**Ответственность:**
- Получает задачу от CEO
- Определяет стратегию тестирования
- Распределяет тест-кейсы между тестировщиками
- Контролирует покрытие тестами
- Утверждает готовность к релизу
- Отчитывается CEO

**Скилы:** test_strategy, test_planning, quality_metrics, team_coordination, release_approval

### Сотрудники

#### 🧪 Manual Tester

**Роль:** Ручное тестирование, исследовательское тестирование.

**Скилы:** test_case_design, exploratory_testing, bug_reporting, regression_testing, usability_testing

#### 🤖 Automation Tester

**Роль:** Автоматизация тестирования, CI-интеграция.

**Скилы:** test_automation, selenium_playwright, pytest_jest, ci_cd_integration, performance_testing
